The basics?

SOPAJS is hosted by Google's AppEngine, so we need to use their servers to handle SSL connections.

Just include the following somewhere in the ⟨head⟩ of your site (note: using "//" instead of "http://" is a way of supporting both HTTP and HTTPS): 

{{''|forceescape}}

Options

The SOPAJS script accepts a number of options:

background_color
The background color for the header. Use valid HTML colors *without* the hash (#). e.g. 888 (not #888).
topic
The topic to display. Valid values: politics, media, companies.
